# Siri Dhanya Hub 🌾

Siri Dhanya Hub is a multilingual Android application focused on millet awareness, healthy recipes, mandi price tracking, and direct farmer connectivity.

Built using Kotlin and Jetpack Compose, the application promotes healthy millet-based lifestyles while supporting local farmers and regional language accessibility.

---

## Features

- 🌾 Millet Information & Health Benefits
- 📈 Live Mandi Watch UI
- 🛒 Direct Farmer Marketplace
- 🍲 Recipe Lab with Multiple Categories
- 💾 Save Recipes Feature
- 🌐 English & Kannada Language Support
- 📱 Modern Jetpack Compose UI
- 🔎 Search Functionality
- 👨‍🌾 Seller Detail Pages
- 📊 Dynamic Market Trends UI
